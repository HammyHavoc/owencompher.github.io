Instructions for downloading the mods for use on oServer:

Download the Forge installer for 1.18 (38.0.17)
Run the .jar and choose install to create the Forge profile in your launcher.
Download this .zip archive and extract the .jar mods into your .minecraft/mods folder.
Start the Forge profile in the launcher and verify that the Simple Voice Chat mod was loaded.
You should now be able to use proximity voice on the server. 'v' is the default keybind to open the voice chat settings.
